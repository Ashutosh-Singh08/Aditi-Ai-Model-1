# Aditi - Personal AI Assistant

Aditi is a personal AI assistant project built by Ashutosh Singh.  
It is designed to chat like a friendly AI companion, remember useful details, respond with voice, and work through a modern web interface.

---

# Features

- AI chat using Ollama local models
- React frontend with modern chat UI
- Node.js + Express backend
- MongoDB database integration
- Voice input using browser speech recognition
- Text-to-speech response support
- Memory system for remembering user preferences
- Chat history storage
- Human-like assistant personality

---

# Tech Stack

## Frontend
- React
- JavaScript
- CSS
- Web Speech API

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Ollama API
